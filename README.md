# CipherStack

Node-based cascade encryption builder for Vyrothon Hackathon Stage 1.

**Muhammad Maaz Ahmad**
mmaaz990ahmad@gmail.com

---

## Tech Stack

- **Next.js 16** — App Router, React 19
- **TypeScript**
- **Zustand** — state management
- **React Flow (@xyflow/react)** — node-based canvas
- **Tailwind CSS v4**
- **Lucide React** — icons
- **Poppins + JetBrains Mono** — typography

## Run locally

```bash
npm install
npm run dev
```
